### 🚀 Featured Engineering Projects

#### 🍎 [NutriScale - IoT AI Smart Meal Tracker](https://github.com/nabeelsyed14/nutriscale)
A full-stack IoT system that automates dietary tracking using embedded hardware and grounded AI.
- **Hardware:** Raspberry Pi 5 edge processing, HX711 + load cell, Pi Camera Module v2.
- **Software:** Flask + SocketIO backend, SQLite, Vision-Grounding framework.
- **Impact:** Uses a Random Forest model to predict daily health trends, combining physical measurement with AI analysis to prevent hallucinations.

#### 🧬 [BioSync - AI-Powered Mobile Health Companion](https://github.com/nabeelsyed14/biosync)
A context-aware health monitoring system that integrates biometric and environmental data.
- **Architecture:** Cross-platform mobile app (Capacitor/Kotlin) + Raspberry Pi IoT pipeline.
- **Data:** MQTT/HTTP real-time data transfer with edge-first cloud logging via Supabase.
- **ML:** Random Forest & Gradient Boosting models predicting a Vitality Index with ~94% accuracy.

#### 💡 LUMISHADE - Sleep Smart Room Automation
An embedded system for autonomous room control based on ambient light and motion.
- **Tech:** STM32 Nucleo-F429ZI, PIR sensors, servo motors, real-time control logic.
- **Features:** Sub-2 second response times, fail-safe mechanisms, and ultra-low power sleep modes.
